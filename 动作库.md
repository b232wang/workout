# 动作库

每个动作的详细文字要领。主动作详述,备用动作附在同一家族下、注明区别。

---

## 推 Push

### 胸

**上斜哑铃卧推**(上胸主力)
椅背调 30°,肘略低于肩、在身体两侧,下放到胸肌充分拉伸再推起,顶端不完全靠拢以保持张力。
- 上斜史密斯卧推:轨迹固定更稳,适合冲重量
- 平板哑铃卧推:整体胸,幅度比杠铃大

**坐姿器械推胸**(中下胸)
把手对齐乳头线,推到接近伸直但不锁死肘关节,慢放感受胸部拉伸。
- 史密斯平板卧推:固定轨迹,可独立冲重量
- 双杠臂屈伸:身体前倾偏胸,身体直立偏三头

**绳索夹胸(高位往低)**(胸,拉伸位)
滑轮调高,身体微前倾,手走弧线在腹前交叉,底部充分拉开、顶峰挤压一秒。
- 蝴蝶机夹胸:轨迹固定,孤立感强
- 上斜哑铃飞鸟:针对上胸,注意肘微屈别变推

### 肩

**哑铃肩推**(前束 / 整体)
坐姿挺背,肘在手腕正下方,推到接近锁定,全程不耸肩。
- 史密斯肩推 / 坐姿器械推举:轨迹固定,更易专注发力

**绳索侧平举**(中束,决定肩宽)
滑轮放最低,身体侧对,小重量、肘微屈,向侧上抬到肩高,底部多停留拉长。**别甩别借力**。
- 哑铃侧平举 / 器械侧平举:同理,器械更稳

**绳索面拉**(后束)
滑轮齐脸高,绳索拉向额头,肘高于手,顶峰外旋挤后束。
- 反向蝴蝶机 / 俯身哑铃飞鸟:同练后束,注意别耸肩

### 三头

**过顶绳索臂屈伸**(长头,拉伸位)
背对滑轮,绳索过头,大臂贴耳固定,在长头被拉长的位置伸肘。长头增长效果好。
- 仰卧哑铃过顶臂屈伸 / 双杠臂屈伸:同样强调拉长位

**绳索下压**(三头)
大臂夹紧身体、只动小臂,下压到完全伸直,慢回。
- 器械臂屈伸 / 窄握史密斯卧推:复合代偿,可冲重量

## 拉 Pull

### 背

**宽握高位下拉**(背阔宽度)
握距略宽于肩,顶部让肩胛充分上伸,沉肩带肘拉到锁骨,不后仰借力。
- 引体向上:自重版本,EMG 背阔激活极高
- 单臂绳索下拉:单侧幅度更大

**坐姿绳索划船**(中背厚度)
挺胸坐直,肘贴身体后拉到腹部,挤肩胛,不耸肩、不弓背甩腰。
- 胸支撑器械划船 / 哑铃单臂划船:减少下背代偿

**直臂下压**(背阔孤立)
手臂保持伸直,以背阔发力把杆下压到大腿前,顶部充分拉伸背阔。
- 器械下拉 / 俯身哑铃划船:替代选择

**反向蝴蝶机**(后束)
坐姿胸靠垫,手臂外展画弧挤后束,全程别耸肩。

### 二头

**上斜哑铃弯举**(拉长位)
椅背后躺,手臂自然后展(二头被拉长),弯举时不晃肘。拉长位有长度优势。
- 坐姿哑铃弯举 / 绳索弯举:常规角度

**锤式弯举**(肱肌 / 前臂)
中立握(掌心相对),不晃身,主练肱肌和前臂、增手臂围度。
- 绳索锤式 / 哑铃交替弯举:同理

## 腿 Legs

### 股四头

**哈克深蹲**
背贴靠垫,脚与肩同宽稍靠下,全程控制下蹲到大腿水平以下再蹬起,膝盖跟脚尖方向。
> 哈克 vs 杠铃深蹲:哈克轨迹固定、不用自己平衡,能更安全专注地轰炸股四头、下背压力小,适合 PF 和增肌;杠铃深蹲是自由重量,练得更全面但门槛高,且 PF 一般没有深蹲架。
- 倒蹬腿举 / 史密斯深蹲:同样固定轨迹的股四头复合动作

**腿屈伸**(股四孤立)
坐姿脚背勾住,伸到顶峰挤压股四一秒,慢放。
- 西斯深蹲 / 单腿腿举:孤立股四的替代

### 腘绳

**罗马尼亚硬拉(哑铃 / 史密斯)**
屈髋把臀往后送,腿微屈,背挺直,重量贴腿下放到腘绳充分拉伸,感受拉长再起。
- 坐姿腿弯举 / 杠铃 RDL:替代选择

**坐姿腿弯举**(腘绳,拉长位)
屈髋坐姿(腘绳处于拉长位,研究显示比俯卧增长多 50%+),脚跟向臀部勾,顶峰挤压。
- 俯卧腿弯举 / 北欧挺:替代选择

### 臀

**臀推**
上背靠凳,杠 / 垫压髋,脚跟发力顶起,顶峰夹臀,下巴微收、别塌腰。
- 器械臀推 / 高箱 step-up:替代选择

### 小腿

**站姿提踵**
底部脚跟充分下沉拉伸,踮到最高停一秒再慢放。站姿(伸膝)比坐姿更练腓肠肌。
- 器械提踵 / 单腿哑铃提踵:替代选择
